SIMON GAME

  • Press the ON button to start the gameboard.
  • Press START to begin the game.
  • You'll be presented with a series of random button presses. Press them in the correct order to advance through the game.
  • You win if you pass all 20 levels.
  • Pressing a wrong button while in STRICT mode resets the game back to the first level. If STRICT mode is off, the demo plays again and you can resume with the same level.
  • Pressing the RESET button starts the game all over again.
